## 🖼 ICÔNE APP

### Spécifications Apple
- **Format** : PNG, pas de transparence, pas de coins arrondis (iOS les ajoute)
- **Taille** : 1024 × 1024 px exactement
- **Fichiers à remplacer** :
  ```
  ios/App/App/Assets.xcassets/AppIcon.appiconset/AppIcon-512@2x.png
  ios/App/QuraniWatch Watch App/Assets.xcassets/AppIcon.appiconset/AppIcon-1024.png
  ios/App/App/public/icons/icon-1024.png
  ```
- **Icône Widget** (iOS 18 adaptatif — 3 variantes) :
  ```
  ios/App/QuraniWidgets/Assets.xcassets/AppIcon.appiconset/
  ```
  → Light mode, Dark mode, Tinted (même logo sur fond adapté)

### Concept recommandé
- **Fond** : crème/ivoire (#F0EDE6) tel que le logo — élégant et premium
- **Marque** : calligraphie noire centrée
- **Style** : minimaliste, prestige — se démarque des icônes Islam standard

### Procédure d'import (quand tu as le fichier)
```bash
# Étape 1 : vérifier dimensions
sips -g pixelWidth -g pixelHeight ~/Downloads/qurani-logo.png

# Étape 2 : redimensionner si nécessaire (macOS built-in)
sips -z 1024 1024 ~/Downloads/qurani-logo.png \
  --out ios/App/App/Assets.xcassets/AppIcon.appiconset/AppIcon-512@2x.png

# Étape 3 : copier pour Watch
cp ios/App/App/Assets.xcassets/AppIcon.appiconset/AppIcon-512@2x.png \
   "ios/App/QuraniWatch Watch App/Assets.xcassets/AppIcon.appiconset/AppIcon-1024.png"

# Étape 4 : icônes PWA
sips -z 512 512 ~/Downloads/qurani-logo.png --out ios/App/App/public/icons/icon-512.png
sips -z 192 192 ~/Downloads/qurani-logo.png --out ios/App/App/public/icons/icon-192.png
cp ~/Downloads/qurani-logo.png ios/App/App/public/icons/icon-1024.png
```

## 🛠 GUIDE XCODE — ÉTAPE PAR ÉTAPE

### Prérequis
- [ ] Nouveau logo 1024×1024 importé dans Xcode (voir procédure ci-dessus)
- [ ] `npm run cap:sync` lancé depuis `/Users/kam/claude/ayat-static`
- [ ] Xcode 16+ ouvert

### Build & Archive
1. Ouvrir `ios/App/App.xcworkspace` dans Xcode
2. Schéma : **App** | Destination : **Any iOS Device (arm64)**
3. **Product > Clean Build Folder** (⇧⌘K)
4. **Product > Archive** (⇧⌘B puis archive)
5. Dans l'**Organizer** : clic droit sur l'archive > **Distribute App**
6. **App Store Connect** > **Upload** > Suivre les étapes

### App Store Connect
1. Aller sur [appstoreconnect.apple.com](https://appstoreconnect.apple.com)
2. **Mes apps > Qurani > + Nouvelle version** : 3.0
3. Remplir :
   - Nom, Sous-titre, Description (copier depuis ce fichier)
   - Keywords
   - What's New
4. **Ajouter les screenshots** (captures depuis le simulateur iPhone 15 Pro Max)
5. **Soumettre pour examen**
