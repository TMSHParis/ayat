1. Vue d'ensemble

Qurani est une application iOS complete pour la lecture, la memorisation et la pratique quotidienne du Coran. Concue avec un design minimaliste et immersif, elle accompagne l'utilisateur dans sa spiritualite au quotidien : lecture du Coran, invocations, horaires de priere, memorisation, et bien plus.

Points forts Aucun compte requis. Aucune publicite. Aucun tracking. Toutes les donnees sont stockees localement sur l'appareil. L'app fonctionne entierement hors-ligne (sauf Mawaqit pour les horaires de mosquee). Disponible sur iPhone, iPad et Apple Watch.

3. Toutes les fonctionnalites

3.1 — Lecture du Coran

  • Lecture verset par verset avec navigation par swipe (gauche/droite)
  • 114 sourates completes en ecriture Uthmani
  • 4 modes de lecture : Minimal, Minimal couleurs, Tajwid, Tajwid couleurs
  • 3 langues : Arabe, Arabe + Francais, Francais
  • 4 themes visuels : Jour, Sepia, Nuit, Auto (detection systeme)
  • 3 tailles de texte : S, M, L avec boutons +/-
  • Marques de waqf (pause) en mode Tajwid
  • Tafsir (commentaire) par appui long sur chaque verset (1.2s)
  • Lecture libre ou lecture guidee (Khatm)

3.2 — Khatm (Lecture complete du Coran)

Systeme complet pour terminer la lecture du Coran avec 5 rythmes au choix :

RythmePortion quotidienneDuree
1 Juz par jour20 pages30 jours
1 Hizb par jour10 pages60 jours
3 Rub par jour7,5 pages80 jours
2 Rub par jour5 pages120 jours
1 Rub par jour2,5 pages240 jours
  • Portions quotidiennes calculees automatiquement
  • Hero card avec compte a rebours "X jours jusqu'au khatm"
  • Liste immersive des sourates avec photos de fond
  • Reader dedie : une sourate a la fois, bouton "Completer" en bas
  • Barre de progression live au scroll (3px, z-index 9999)
  • Historique complet des sessions (heure, sourate, versets debut/fin)
  • Dropdown iOS (position fixed) pour Historique et Suppression
  • Sheet de confirmation pour suppression avec recapitulatif

  • Memorisation verset par verset, mot par mot

  • 5 prieres affichees : Fajr, Dhuhr, Asr, Maghrib, Isha

  • Overlay priere immersif style Behold (117 photos de fond)
  • Footer avec QIBLA, bouton fermeture (X), REGLAGES

4. Adaptation iPad

L'interface est entierement responsive pour iPad avec 2 breakpoints CSS :

BreakpointAppareilAdaptations
768pxiPad (portrait)Padding 10%, max-width 640px, fonts agrandies
1024pxiPad Pro (paysage)Padding 12%, layouts centres, marges elargies

Elements adaptes

  • Dashboard : fond plein ecran avec padding proportionnel
  • Du'a liste : grille centree avec max-width 640px
  • Du'a detail : padding 10%, fonts agrandies
  • Khatm : layout responsive avec hero et liste immersive
  • Tous les overlays : centres avec marges adaptees
  • Tab bar : largeur et espacement ajustes
  • Orientation : toutes orientations supportees (portrait + paysage)
iPhone vs iPad iPhone : portrait uniquement. iPad : toutes orientations avec adaptation CSS dynamique.

6. Design & Direction artistique

Philosophie

Design minimaliste et immersif inspire par les apps modernes (Behold, Apple Weather). Experience de lecture pure et sans distraction, avec une esthetique soignee qui respecte le contenu sacre.

Overlay priere (style Behold)

  • 117 photos de fond immersives haute qualite
  • Footer avec QIBLA, bouton X (fermeture), REGLAGES
  • Bouton X : fond semi-transparent rond, trait epais (2.5px), blanc lumineux
  • Horaires en police Space Mono
  • Transitions fluides entre les vues

Themes visuels

ThemeDescription
JourFond clair, texte sombre
SepiaFond beige chaud, confort de lecture prolongee
NuitFond sombre, texte clair, reduit la fatigue oculaire
AutoDetection automatique du mode sombre/clair du systeme iOS

Polices

  • Space Mono (Regular + Bold) : overlay priere, numeros
  • NotoNaskhArabic : texte arabe principal
  • ScheherazadeNew : texte arabe alternatif
  • -apple-system : interface generale

Details UX

  • Scrollbars masquees sur toute l'app (scroll fluide, barre invisible)
  • Navigation par swipe naturel
  • Splash screen avec detection instantanee du theme (zero flash)

8. Build & Deploiement

Workflow

  1. Modifier app.js, style.css, index.html
  2. Incrementer la version du Service Worker dans sw.js
  3. Executer npm run cap:sync
  4. Xcode : scheme "App" > destination "Any iOS Device (arm64)"
  5. Product > Archive
  6. Distribute App > App Store Connect
  7. Sur App Store Connect : associer le build et soumettre
