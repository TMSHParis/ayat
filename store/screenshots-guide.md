# Guide des captures d'écran pour les stores

## Apple App Store (obligatoire)

### iPhone 6.7" (iPhone 15 Pro Max) — OBLIGATOIRE
- Dimensions : 1290 x 2796 px (portrait)
- Format : PNG ou JPEG
- Minimum 3 captures, maximum 10

### iPhone 6.5" (iPhone 14 Plus)
- Dimensions : 1284 x 2778 px (portrait)

### iPad 12.9" (si vous supportez iPad)
- Dimensions : 2048 x 2732 px (portrait)

## Google Play Store (obligatoire)

### Téléphone
- Dimensions : entre 320px et 3840px (côté minimum)
- Ratio recommandé : 9:16 (portrait)
- Dimensions recommandées : 1080 x 1920 px
- Format : PNG ou JPEG (24 bits, pas d'alpha)
- Minimum 2 captures, maximum 8

### Tablette 7" (recommandé)
- Dimensions : 1080 x 1920 px

### Tablette 10" (recommandé)
- Dimensions : 1080 x 1920 px

## Comment créer les captures

### Méthode rapide : Simulateur Xcode
1. Ouvrir le projet dans Xcode
2. Lancer sur un simulateur iPhone 15 Pro Max
3. Prendre des captures avec Cmd+S dans le simulateur
4. Les captures seront aux bonnes dimensions automatiquement

### Captures à prendre (5 écrans)

1. **Écran principal** — Afficher un beau verset (ex: Sourate Al-Fatiha, verset 1)
   → Thème Jour (fond blanc)
   → Texte "Lisez le Coran verset par verset"

2. **Progression** — Après avoir lu quelques versets pour montrer les barres
   → Barre du jour à ~40%, barre khatma à ~15%
   → Texte "Suivez votre avancement jour après jour"

3. **Thème Nuit** — Même écran en mode sombre
   → Texte "3 thèmes pour votre confort : Jour, Sépia, Nuit"

4. **Liste des sourates** — Overlay des sourates ouvert
   → Texte "Parcourez librement les 114 sourates"

5. **Réglages** — Page des réglages ouverte
   → Texte "Un rappel quotidien pour ne rien oublier"

### Outil recommandé pour ajouter du texte
- Screenshots.pro (gratuit en ligne)
- Ou simplement Keynote / PowerPoint avec les bonnes dimensions
